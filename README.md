# Lashesforgirlies Management Bot
## Author: Huy Hoang Nguyen

Lashesforgirlies Management Bot is an internal business tool developed to support **Lashesforgirlies**, a small eyelash salon. 
The project is currently in active use and helps optimize daily management workflows.

The main features include:
- **View Schedule:** Display upcoming appointments to help staff track bookings easily.
- **Income Calculation:** Calculate total income, costs, and profit by month or for all time.
- **Customer History:** Lookup past services and payments for individual customers.

## Technology Stack
This project uses the following Python libraries and services:
- **pyTelegramBotAPI**￼ (~4.29.1): For interacting with the Telegram Bot API.
- **python-dotenv￼** (~1.2.1): Load environment variables from .env for local development.
- **Flask**￼ (~3.1.2): Web framework used to receive webhook requests from Telegram.
- **gspread**￼ (~6.2.1): Google Sheets API wrapper for reading/writing spreadsheet data.
- **google￼** (~3.0.0): Google APIs client for authentication and service account management.

Other tools and platforms:
- **Railway**￼: Cloud platform used to deploy the bot.
- **Google API**: Used for accessing and managing Google Sheets data.
